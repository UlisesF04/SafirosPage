## What is this project?

Safiro's is a website built for a real local clothing store. The client needed an online
presence to showcase their catalog and attract new customers. The project includes a
landing page presenting the brand and a product catalog page where visitors can browse
the available items.

This was my first project built for a real client outside of personal practice,
which introduced me to working with actual requirements, deadlines, and feedback cycles.

## Features

- Responsive landing page presenting the brand
- Product catalog with clothing items
- Mobile-friendly layout
- Deployed and live on Netlify

## Tech Stack

- **HTML5** — semantic page structure
- **CSS3** — custom styling and responsive layout
- **JavaScript** — dynamic catalog rendering and UI interactions
- **Netlify** — static site hosting and deployment

## What I learned

Building for a real client is a different experience from personal projects.
Requirements change, feedback is subjective, and the final result has to work
for someone who isn't a developer. This project taught me to translate
non-technical requirements into working features and to communicate clearly
about what is and isn't feasible.
